前言

在Linux下开发Android某些情形下比windows方便点，特别是涉及到AOSP或Framwork相关的，以下均在Ubuntu 20上测试。

环境搭建

  1. JDK安装 一般是使用apt安装 
    sudo apt install openjdk-8-jdk
sudo apt install openjdk-11-jdk

    若安装openjdk-8-jdk失败，可以参考这里，手动安装 切换命令版本

    sudo update-alternatives --config java
sudo update-alternatives --config javac
  2. Android SDK安装 下载SDK相关管理工具和脚本 
    mkdir ~/android-sdk & cd ~/android-sdk
wget https://dl.google.com/android/repository/commandlinetools-linux-6858069_latest.zip
unzip commandlinetools-linux-6858069_latest.zip
./cmdline-tools/bin/sdkmanager --sdk_root=<userpath>/android-sdk/ --list
#安装编译工具
./cmdline-tools/bin/sdkmanager --sdk_root=<userpath>/android-sdk/ "platforms;android-30" "build-tools;30.0.3" "cmake;3.18.1" "ndk;21.4.7075529"

    配置SDK环境变量，在~/.bashrc文件后追加

    export ANDROID_SDK_ROOT=<userpath>/android-sdk
export ANDROID_HOME=$ANDROID_SDK_ROOT
export PATH=$PATH:$ANDROID_HOME/tools:$ANDROID_HOME/tools/bin:$ANDROID_HOME/platform-tools

    以上<userpath>是指当前用户目录，比如我的是/home/liubo

gradle编辑出错，可以使用gradle --info --debug --stacktrace打印详细错误

adb相关

  1. adb连接设备报permission错误 
    #将当前用户加入到plugdev组
sudo usermod -aG plugdev $LOGNAME
#安装设备
sudo apt-get install android-sdk-platform-tools-common
#重启
sudo reboot
  2. adb device 报unauthorized，是手机上没有允许调试

编译运行项目

通过上述步骤，已经可以编译运行gradle构建的android项目了

# 拉取项目源码
git clone http://xxx.yy.git
cd <codepath>
./gradlew assemble

如果需要android studio，可以参考以下方式安装

#下载linux版本android studio
wget https://redirector.gvt1.com/edgedl/android/studio/ide-zips/2020.3.1.26/android-studio-2020.3.1.26-linux.tar.gz
#解压 <yourpath>解压缩后的文件存放目录
tar -zxv -f android-studio-2020.3.1.26-linux.tar.gz -C <yourpath>
#在终端中运行
.<yourpath>/android-studio/bin/studio.sh

每次用sh启动不方便，可以在Ubuntu上桌面创建AndroidStudio图标

#进入桌面目录，英语条件下桌面目录的名称可能是Desktop
cd ~/桌面
cat > ~/桌面/android-studio.desktop
#拷贝以下文字，输入到终端，注意<yourpath>替换为自己的AndroidStudio路径
[Desktop Entry]
Encoding=UTF-8
Name=AndroidStudio
Comment=AndroidStudio IDE
Exec=<yourpath>/android-studio/bin/studio.sh
Icon=<yourpath>/android-studio/bin/studio.png
Terminal=false
StartupNotify=true
Type=Application
Categories=Application;Development;

#Ctrl+C 退出后，桌面出现android-studio.desktop文件，右键选择“允许启动”即可
