  • JNI在Android平台的使用

    2020-06-11
    liubo
    Android  Java
    android-NDK  JNI  make  cmake
    • 前言

    前言

    Android开发中，JNI可以用于跨平台，也可以用于提高性能以及代码安全性等，这里也提一下可以使用JNA技术实现C/C++接口的访问，JNA的优点是不需要编写额外的C/C++代码，仅仅是java代码即可完成底层接口的调用，缺点是性能相对较差。

    什么是JNI呢？JNI全称是Java native interface，即为Java与本地代码的通信接口，Android中会通过NDK使用JNI，NDK是什么呢？NDK全称为Native develop kit，即为本地开发工具包，Goole为Android开发者提供了工具包，用于Android平台的JNI开发，该工具包支持Windows和Linux。

  • 创建一个属于自己的博客

    2020-05-24
    liubo
    Blog
    blog  markdown
    • 前言
    • 遇到的问题
    • 参考资料

    前言

    很多人可能都有一种想法，写点东西记录下学习或生活，好记忆不如烂笔头，现在我们用博客来开启文字记忆。

    需要使用到的资源有：GiteePages，jekyll，Theme，主题是一个大牛开源的，只需要根据他的中文文档配置，基本可以完成搭建。

    遇到的问题

    1. 不蒜子的域名过期 根据不蒜子网站说明更改即可。

    2. 百度统计 需要在百度统计官网注册，更改_config.yml文件的配置，需要注意的是若网站1个月没有流量，可能失效。

    3. 文档撰写 文档存放在_posts目录，使用Markdown语言编写，可参考RUNOOB的教程。

    4. 文档摘要 在_config.yml中使用excerpt_separator来标记的，在本项目中使用的是四个连续空行
