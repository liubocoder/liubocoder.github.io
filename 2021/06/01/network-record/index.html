  1. linux tcpdump 
     示例：
 tcpdump -D 列举端口信息
 tcpdump -i eth0 -nn -XX  ip host xxip and port 123
 -i eth0 网卡eth0
 -nn 所有报文转数字
 -XX输出包的头部数据，会以16进制和ASCII两种方式同时输出，更详细
