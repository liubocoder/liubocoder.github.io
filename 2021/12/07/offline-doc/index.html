搭建内网离线文档

2021-12-07
liubo

Blog
blog
  • 前言
  • 使用方法
  • 将离线文档部署到内网

前言

有时候查询某些开源项目的文档时，由于各种原因导致网页加载速度很慢，这里记录一个工具Zeal，可以抓取主流开源项目的文档，用于离线查看

免安装版本下载地址

使用方法

  1. 下载软件包压缩文件
  2. 解压运行 tools->Docsets->Available，选取自己的需要抓取的文档

将离线文档部署到内网

这里以ElasticSearch的文档为例，文档抓取完成后，会放在如下目录

Zeal软件目录\docsets\ElasticSearch.docset\Contents\Resources\Documents

将该目录下所有文件拷贝到要部署的服务器上，例如我这里服务器是Ubuntu，拷贝到/opt/nginx_es/www

创建需要的映射的目录

sudo su
mkdir /opt/nginx_es/conf /opt/nginx_es/logs /opt/nginx_es/www

创建配置文件

cat > /opt/nginx_es/conf
worker_processes  1;

events {
    worker_connections  1024;
}

http {
    include       mime.types;
    default_type  application/octet-stream;
    sendfile        on;
    keepalive_timeout  65;

    server {
        listen       80;
        server_name  localhost;
        location / {
            root   html;
            index  index.html index.htm;
        }
        location /gwsdn/nginxstat {
            stub_status            on;
            access_log              on;
            #auth_basic              "show simple nginx status";
            #auth_basic_user_file  conf/password;
        }
    }
}

运行容器，这里服务器上有占用80端口的程序，所以映射到了8200

docker run -d -p 8200:80 --name es-nginx --restart=always \
  -v /opt/nginx_es/www:/etc/nginx/html \
  -v /opt/nginx_es/conf/nginx.conf:/etc/nginx/nginx.conf \
  -v /opt/nginx_es/logs:/var/log/nginx \
  nginx:latest
